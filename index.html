Literature-as-code

[The Analytical Engine] ...might act upon other things besides number

Augusta Ada King (Née Byron), countess of Lovelace

Besides casting the computer as an universal machine, it's quite clear that there is no big difference between numbers and anything else, as long as you can express them symbolically and their relationship algebraically. There is something about this in Sydney's Paduas novel where Ada Lovelace programs the analytical engine so that it checks "books" and rips apart those that do not conform to a standard. A bit of this in this blog post.. So theoretically you could think about flights of fancy like using text generators, or text "fillers" or any other kind of helpers for the particular dialog style, or book atmosphere, or whatever. And there's no API or SDK

use Text::Markdown;
sub MAIN( $dir = "txt/", $length = 60 ) {
    for $dir.IO.dir(:test(/ ".md" $/)) -> $f {
	my @items =  Text::Markdown::Document.new($f.slurp()).items;
	say ( @items 
	      ==> grep( { $^þ ~~ Text::Markdown::CodeBlock } )
	      ==> map( { .text.split("\n") } )
	      ==> flat()
	      ==> grep( { .chars > $length }  )
	      ==> map( { "‣$f → $^þ\n ⤷ is "~$^þ.chars~" characters long" } )
	      ==> join( "\n" xx 2 ) );	
    }
}
